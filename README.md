# YANBI-ML

Yet Another Naive Bayes Implementation

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'yanbi-ml'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yanbi-ml
